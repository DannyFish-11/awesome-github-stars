## 📖 项目简介

本仓库每天自动收集 **15 个 GitHub 高 star 开源项目**，包括：

- 🔥 **GitHub Trending**：当日最热门的趋势项目
- ⭐ **Top Stars**：历史累计 star 数最高的优质项目

所有项目均为完整开源项目，涵盖多种编程语言和技术领域，适合学习、参考和使用。

```
│       ├── 2026-01-21.md
│       └── ...
└── archive/                     # 历史归档
```

## 📊 收集规则

- **收集频率**：每天自动执行
- **项目数量**：每天 15 个
- **项目标准**：5k+ stars 或当日热门趋势
- **仓库容量**：每个仓库存放约 100 个项目（7 天）
- **更新时间**：每天 UTC+8 时区自动更新

## 📝 项目信息

每个项目包含以下信息：

- 项目名称和链接
- 编程语言
- Star 数和 Fork 数
- 项目简介
- 今日新增 Star 数（如适用）
